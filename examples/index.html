jQuery Hashtager Examples

jQuery Hashtager is a simple plugin for marking up hashtags in content or geting list of hashtags from string.

You can find the newest version of plugin in jQuery Hashtager Github page or download the latest version.

Usage

For start include js file

<script type="text/javascript" src="jquery.hashtager.js"></script>

For simple usage just call .hashtager() on some selector.

<div class="selector">Everybody #love good #hashtags</div>

$('.selector').hashtager();

In return we will have

<div class="selector">Everybody <strong class="hashtager">#love</strong> good <strong class="hashtager">#hashtags</strong></div>

Settings

Parameter Description Default
template Template to replace hashtags <strong class="{CLASS}">#{ANCHOR}</strong>
numericTemplate Custom template for numeric hashtags. If field is empty will be used main template
class {CLASS} will be replaced with this field value hashtager

Also you can call .hashtager() on string and get array of hashtags on return

var hashtagsList = "Everybody #love good #hashtags".hashtager();

Variable hashtagList will contain ["love", "hashtags"]

Settings

Parameter Description Default
type What type of hashtags return.
  • word - only alphanumeric hashtags, starting with letter
  • num - only numeric hashtags
  • all - all type of hashtags
all
withSign Return matches with hash sign or not false
